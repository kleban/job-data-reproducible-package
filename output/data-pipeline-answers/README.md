# Reviewer-Answer Outputs: Data Pipeline

This directory contains the manuscript-ready CSV tables and supporting statistics generated by the public reviewer-answer notebooks.

## Manuscript tables

| Manuscript output | Generated file | Producer |
|---|---|---|
| Table A2, Panel A | `validation/table_a2_panel_a_posting_language.csv` | Notebook 02 |
| Table A2, Panel B | `validation/table_a2_panel_b_matching_method.csv` | Notebook 02 |
| Table A3 | `validation/table_a3_manual_validation_accuracy.csv` | Notebook 03 |
| Table A4 | `validation/table_a4_error_rates_by_language.csv` | Notebook 03 |
| Table A5 | `validation/table_a5_error_rates_by_matching_method.csv` | Notebook 03 |
| Table A6 | `threshold-selection/classification_accuracy_by_threshold.csv` | Notebook 04 |

All table files are semicolon-delimited CSV. Table A2 preserves the display percentages used in the manuscript; the supporting statistics retain directly calculated percentages and counts.

## Supporting outputs

The `validation/` directory also contains population/sample distributions, stratum matrices, hierarchical agreement results, and error-rate tables. See [its README](validation/README.md) for the full inventory.

The Table A6 output reports accuracy for thresholds 0.5–0.9 from included aggregate correct/compared counts. Its record-level limitation is documented in the data and notebook READMEs.
