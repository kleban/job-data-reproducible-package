## Threshold-selection output

The `threshold-selection/classification_accuracy_by_threshold.csv` file reproduces manuscript Table A6. It is generated by `04_compare_extraction_thresholds.ipynb` from the included archived aggregate correct/compared counts and reports accuracy at one decimal place for thresholds 0.5 through 0.9.
